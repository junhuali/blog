发表于 2020-06-10 分类于 Linux Disqus：

Nginx 配置语法

  • 配置文件由指令与指令块组成
  • 每条指令以 ; 结尾，指令与参数以空格符号分隔
  • 指令块以{} 大括号将多条指令组合在一起
  • include 语句允许组合多个配置文件以提升可维护性
  • 使用#符号添加注释，提高可读性
  • 使用$符变量
  • 部分指令的参数支持正则表达式

http配置指令块主要包含下面4个部分

  • http
  • server
  • location
  • upstream

Nginx命令行

  • 格式 nginx -s reload
  • 帮助 -? -h
  • 使用指定的配置文件 -c
  • 指定配置文件指定 -g
  • 指定运行目录 -p
  • 发送信号 -s(立刻停止 stop 优雅退出 quit 重载配置文件 reload 重新开始记录日志文件reopen)
  • 测试配置文件是否有语法错误 -t -T
  • 打印Nginx的版本信息、编译信息等 -v -V
