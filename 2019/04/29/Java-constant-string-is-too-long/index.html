Java 常量字符串过长

发表于 2019-04-29 | 更新于 2019-06-22 | 分类于 Java | 评论数：

问题出现

在IDEA中，字符串长度超过65535，进行打印，IDEA会提示java: 常量字符串过长。使用javac 进行编译也会有类似的提示

解决办法

使用StringBuilder

创建了一个对象，会根据实际的存储的内容分配内存，不会受到String在常量池中最大长度的限制
通过使用StringBuilder解析大文本和大JSON
