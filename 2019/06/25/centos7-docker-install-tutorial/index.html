安装

可以通过多种方式安装Docker CE：

  • 大多数用户通过设置 Dokcer's repositories 的方式安装，方便安装和进行升级，这种方法比较推荐。
  • 有些用过通过下载RPM 包的方式安装，这种方式适合没有办法连接到网络的时候使用。
  • 在开发和测试环境，使用自动脚本进行Dokcer安装
